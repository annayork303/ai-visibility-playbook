# Schema Snippets

Copy-paste-ready JSON-LD structured data (Element 18 of [the playbook](../PLAYBOOK.md)).

| File | Use on |
|---|---|
| [article-schema.html](article-schema.html) | Every blog post or article page |
| [faq-schema.html](faq-schema.html) | Pages with a Q&A / FAQ section |

Add the snippet inside your page `<head>`, fill in your values, then validate at [search.google.com/test/rich-results](https://search.google.com/test/rich-results).
